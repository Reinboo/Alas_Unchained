# Alas Unchained

React+Firebase CMS for submitting university assignments

## Getting Started

### Prerequisites

You need to have nodejs(10+), npm(6.4.1+) and git installed

```
apt install nodejs npm git
```

### Installing

- Clone repo:

```
git clone https://github.com/Reinboo/Alas_Unchained
```

- Install npm dependencies:

```
npm install
```

You can run dev server with 

```
npm start
```

or get production build with 

```
npm run build
```
